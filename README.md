# Data-sphere
Complete end to end process for database interaction with NLP to SQL conversion, along with intelligent ML model recomendation based on Dataset and automated data cleaning software.
This project provides a complete end-to-end pipeline that enables users to:

Query databases using Natural Language (NLP-to-SQL)
Automatically clean and preprocess datasets
Analyze dataset structure and characteristics
Recommend best Machine Learning models
Train, evaluate, and deploy selected models
Visualize insights from structured data

The system integrates:

Database interaction
Natural Language Processing
Automated Data Cleaning
Meta-Learning Based Model Recommendation
Machine Learning Pipeline Automation
